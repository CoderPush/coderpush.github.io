Who we are#

CoderPush is a studio of engineers and designers to collaborate with clients for great software, teams, and processes.

We set a very high expectation of client satisfaction for the long term. Some of our clients have volunteered their time to leave testimonials on Clutch.

We are looking for someone with#

  • 2+ year experience of development with Ruby on Rails, or similar fullstack (MVC) experience that can convert to the Ruby ecosystem fast.
  • Good understanding of SOLID principles, common design patterns and best practices
  • Experience interacting with SQL databases, such as performance tuning and security
  • Experience with complicated system architecture which includes API linkage with external systems
  • Some experience with writing unit tests and a strong desire to write tests
  • Good knowledge or awareness of web security
  • Follow CI/CD and produce well written Pull Requests daily
  • Good English communication skills
  • Experience of leading a project as a team leader
  • Strong sense of ownership

It will be a big plus if you have these experience:

  • Experience with application performance monitoring and optimization
  • Experience with Amazon Web Services
  • Experience with service operation on production
  • Experience with frontend development (TypeScript, ReactJS/VueJS)
  • Experience of training junior/middle members

What you will work on#

  • Design and develop services which can handle a massive amount of data
  • Build API backends satisfying functional and non-functional requirements
  • Work with both REST and GraphQL APIs and optimize them
  • Write clean and maintainable code
  • Follow best practices through static code analysis
  • Monitor application performance and fix system failures
  • Can learn fast from our senior engineers, especially in Ruby/Rails/DevOps Requirements

Benefits#

  • Salary $1.5k-$2.5k gross depending on experience.
  • Performance-based bonuses tied to project deliverables.
  • Sponsorships to take part in sports and tech activities.
